# cubekit

A modular Python toolkit for cube simulation, human-style solving workflows, validation, and visualization.

## Status

Early development version (v0.0.1).

This package is being actively developed to support extensible cube modeling and human-style solving pipelines.

## Goals

- Modular cube simulation architecture
- Human-style solving workflows
- Extensible solver pipelines
- Validation and testing utilities
- Visualization support (planned)
